# YourPlaceGH

Lean WordPress real estate theme for yourplacegh.com (Accra, Ghana).

## Design source
- `yourplacegh-home.html` : standalone homepage design mockup. Reproduces the live "Quarter" homepage layout on the locked identity (Poppins 700 display / Nunito Sans 400 body, orange #F15A2B on dark slate #242C34). Built lean: pure HTML/CSS plus one small vanilla-JS slider, no Bootstrap/Elementor/RevSlider. This file is the source of truth for the theme `front-page.php` conversion.
